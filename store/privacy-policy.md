# Privacy Policy — GitLab Ninja

**Last updated:** 2026-03-13

## Data Collection

GitLab Ninja does **not** collect, transmit, or store any data on external servers. There are no analytics, telemetry, or tracking of any kind.

## Data Stored Locally

The extension stores the following data locally on your device using the browser's built-in storage API (`chrome.storage`):

- **GitLab instance URL** — The URL of your GitLab instance (e.g. `https://gitlab.com`)
- **Personal Access Token** — Your GitLab API token, used to authenticate API requests
- **Feature preferences** — Your toggled settings (e.g. auto-assign on/off, reminder intervals)
- **Theme preference** — Light or dark mode selection

This data never leaves your browser.

## Network Requests

The extension makes API requests **only** to the GitLab instance URL you configure. These requests are used to:

- Fetch issue and time tracking data for board display
- Create issues and set time estimates when you use the popup
- Check daily time logged for reminder notifications

No requests are made to any other server.

## Third Parties

GitLab Ninja has zero third-party runtime dependencies. No external scripts, SDKs, or services are loaded.

## Data Deletion

Uninstalling the extension removes all locally stored data. You can also clear your data at any time from the extension's Settings page.
